# 🔢 Basic Sorting Algorithms (Java)

This repository contains implementations of fundamental sorting algorithms in Java. It is designed for beginners to understand sorting concepts clearly and practice DSA for coding interviews.

---

## 📚 Topics Covered

- What is Sorting?
- Bubble Sort
- Selection Sort
- Insertion Sort
- Counting Sort
- Inbuilt Sorting Methods

---

## 💻 Features

- ✅ Simple and clean Java code
- ✅ Step-by-step understanding
- ✅ Beginner-friendly explanations
- ✅ Practice questions included
- ✅ Solutions for better learning

---

## 🧠 Algorithms Included

| Algorithm        | Time Complexity (Worst) | Space Complexity |
|------------------|------------------------|------------------|
| Bubble Sort      | O(n²)                  | O(1)             |
| Selection Sort   | O(n²)                  | O(1)             |
| Insertion Sort   | O(n²)                  | O(1)             |
| Counting Sort    | O(n + k)               | O(k)             |
| Inbuilt Sort     | O(n log n)             | Depends          |
